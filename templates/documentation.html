A Web-based Tool Utilising Deep Neural Network to Predict Significant Enhancer Signals

How it works

EPNET is adapted from the original PNET model which predicts prostate cancer disease state.

The network is built from the Reactome Pathways database a curated and peer-reviewed pathway database containing information on biological interactions that are grouped into hierarchical pathways. A set of relevant pathways were extracted from the database to generated a layered network representing known parent-child relationships between the pathways.

A matrix containing 0s and 1s representing enhancer-promoter interactions is split into training, validation and testing sets and then fed into the model to train its prediction. The trained model is able to predict whether a given enhancer is a true enhancer.
